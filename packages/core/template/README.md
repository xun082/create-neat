## 如何运行 & 打包

### 启动项目

在开发模式下运行 
```
${packageManager} run start
```
打开 [http://localhost:3000](http://localhost:3000) 以在浏览器中查看。
如果您进行编辑，页面将重新加载
您还会在控制台中看到任何 lint 错误。

### 打包项目
```
 ${packageManager} run build
```
运行后会将项目打包，默认生成名为 dist 的文件
